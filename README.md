## Run Locally

Clone the project

```bash
  git clone https://github.com/jane-does-coding/zodiac-sign-calculator
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm i
```

Run

```bash
  npm run dev
```
